# Aegis

Aegis is a security-first, non-custodial desktop wallet built in Rust for the Solana ecosystem.

## MVP Scope

- Desktop app using Tauri
- Solana-only
- Non-custodial
- Local encrypted wallet storage
- SOL and SPL token support
- No NFT support in MVP
- No custom backend in MVP

## Development Rule

Use well-known, actively maintained crates whenever possible. Do not build custom cryptography.
